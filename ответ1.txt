 var a = 1, b = 1, c, d;
c = ++a; alert(c); // 2
Префикская форма инкремента сначал увеличивает значение на единицу, а затем присваивает его переменной 
d = b++; alert(d); // 1
Постфиксная форма инкремента сначала присваивает значение, а затем увеличивает его на единицу 
c = (2+ ++a); alert(c); // 5 
'a' на момент выполнения равно 2 и увеличивается на 1 префиксно, получаем с = 2 + 3 
d = (2+ b++); alert(d); // 4 
'b' на момент выполнения равно 2, но увеличивается на 1 постфиксно (то есть в действии участвует его старое значение), получаем d = 2 + 2 
alert (a); // 3 
Изначально было 1 и дважды увеличивалось на 1 при операциях
